# 📊 FundSight — Mutual Fund Analytics Platform

A full-stack analytics platform for mutual fund data ingestion, processing, SQL-based analysis, and interactive dashboards.

---

## 🗂️ Project Structure

```
fundsight-mutual-fund-analytics/
│
├── data/
│   ├── raw/           # Raw ingested data (CSVs, JSON, API dumps)
│   └── processed/     # Cleaned & transformed datasets
│
├── etl/               # Extract, Transform, Load pipeline scripts
├── sql/               # SQL queries, schemas, and stored procedures
├── notebooks/         # Jupyter notebooks for EDA and analysis
├── dashboard/         # Dashboard app (Streamlit / Dash / Power BI)
├── reports/           # Generated reports, charts, and exports
└── docs/              # Project documentation and architecture notes
```

## 🛠️ Tech Stack

| Layer       | Tool / Library              |
|-------------|-----------------------------|
| Ingestion   | Python, Requests, yfinance  |
| ETL         | Pandas, SQLAlchemy          |
| Storage     | PostgreSQL / SQLite         |
| Analysis    | SQL, Jupyter Notebooks      |
| Dashboard   | Streamlit / Plotly Dash     |
| Reports     | Matplotlib, Seaborn         |
